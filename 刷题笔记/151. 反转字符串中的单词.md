# [151. 反转字符串中的单词](https://leetcode.cn/problems/reverse-words-in-a-string/)

中等



给你一个字符串 `s` ，请你反转字符串中 **单词** 的顺序。

**单词** 是由非空格字符组成的字符串。`s` 中使用至少一个空格将字符串中的 **单词** 分隔开。

返回 **单词** 顺序颠倒且 **单词** 之间用单个空格连接的结果字符串。

**注意：**输入字符串 `s`中可能会存在前导空格、尾随空格或者单词间的多个空格。返回的结果字符串中，单词间应当仅用单个空格分隔，且不包含任何额外的空格。

 

**示例 1：**

```
输入：s = "the sky is blue"
输出："blue is sky the"
```

**示例 2：**

```
输入：s = "  hello world  "
输出："world hello"
解释：反转后的字符串中不能存在前导空格和尾随空格。
```

**示例 3：**

```
输入：s = "a good   example"
输出："example good a"
解释：如果两个单词间有多余的空格，反转后的字符串需要将单词间的空格减少到仅有一个。
```

 

**提示：**

- `1 <= s.length <= 104`
- `s` 包含英文大小写字母、数字和空格 `' '`
- `s` 中 **至少存在一个** 单词



 

**进阶：**如果字符串在你使用的编程语言中是一种可变数据类型，请尝试使用 `O(1)` 额外空间复杂度的 **原地** 解法



```c++
class Solution {

public:

  string reverseWords(string s) {

​    stringstream ss(s);

​    vector<string> words;

​    string w;

​    string res;

​    while(ss>>w) words.push_back(w);

​    reverse(words.begin(),words.end());

​    for(int i=0;i<words.size()-1;i++)

​    {

​      res = res+words[i] + " ";

​    }

​    res = res+words[words.size()-1];

​    return res;

  }

}; 
```

#  **stringstream 常用也好用的功能大全**

------

# 1. **把“字符串”当成“输入流”（最常用）**

你已经见过：

```
stringstream ss(s);
string w;
while (ss >> w) { ... }   // 自动按空白分单词
```

特点：

- 自动跳过空白字符
- 提取一个 token（可为 string / int / float / double 等）

适合：
 ✔ 切分字符串
 ✔ 从字符串中解析数字
 ✔ 解析配置文件 / 指令

------

# 2. **从字符串中解析数字（简直神器）**

例子：

```
string s = "123 45.6 -7";
stringstream ss(s);

int a;
double b, c;

ss >> a >> b >> c;

cout << a << ", " << b << ", " << c << endl;
// 输出：123, 45.6, -7
```

不需要你手动 `stoi()`、`stod()`，而且自动跳空格。

------

# 3. **拼接字符串（像 stringbuilder）**

用 `<<` 往里写：

```
stringstream ss;
ss << "Player HP: " << 100 << ", Score: " << 2500;
string s = ss.str();
```

结果：

```
"Player HP: 100, Score: 2500"
```

比 `string + int + ...` 更安全、可读性更好。

适合：
 ✔ 日志系统
 ✔ 打印 debug 信息
 ✔ UI 文本组合

------

# 4. **格式化读写（类似 printf，但更安全）**

例如写小数点精度：

```
stringstream ss;
ss << fixed << setprecision(2) << 3.14159;

cout << ss.str();  // 3.14
```

等价于：

- `fixed`：固定小数格式
- `setprecision(n)`：保留 n 位小数
- `setw(n)`：设置字段宽度
- `setfill(c)`：设定填充值

可组合使用：

```
stringstream ss;
ss << setw(10) << setfill('0') << 42;
cout << ss.str();  // "0000000042"
```

------

# 5. **清空并重用 stringstream**

你可以把它当循环 buffer 使用：

```
stringstream ss;

ss << "hello";
cout << ss.str() << endl;

ss.str("");   // 清空内容
ss.clear();   // 重置状态标志（非常重要）

ss << "world";
cout << ss.str() << endl;
```

------

# 6. **读取一整行**

比如按行处理 CSV：

```
string s = "apple,100,red";
stringstream ss(s);

string item;
while (getline(ss, item, ',')) {
    cout << item << endl;
}
```

输出：

```
apple
100
red
```

`getline(ss, item, ',')`：以 `,` 为分隔符，比 `ss >>` 灵活。

------

# 7. **按照特定格式解析结构体（非常高级）**

例子：解析一个向量：

```
"(1.2, 3.4, 5.6)"
```

可以这样：

```
string s = "(1.2, 3.4, 5.6)";
stringstream ss(s);

char c;
float x, y, z;
ss >> c >> x >> c >> y >> c >> z >> c;

cout << x << " " << y << " " << z;
```

只要输入格式固定，你可以按顺序解析任意内容。

------

# 8. **stringstream 还能和文件配合使用**

读文件内容后进行进一步解析：

```
ifstream fin("config.txt");
stringstream buffer;
buffer << fin.rdbuf();

string content = buffer.str();
```

你就能一次性读取整个文件，然后用 stringstream 再拆解。

------

# 🌈 总结：stringstream 的强项

| 功能                          | 描述                            |
| ----------------------------- | ------------------------------- |
| **自动拆分单词**              | 最适合 split（比手写强太多）    |
| **自动解析数字**              | 等价于 `stoi`、`stod`，但更安全 |
| **拼接字符串**                | 类似 stringbuilder              |
| **可设置格式**                | 支持精度、填充、对齐            |
| **读取任意格式的结构体**      | 按格式符号解析                  |
| **结合 getline 做自定义分隔** | 可以按任意字符切割              |

一句话：

> **stringstream 用来做“字符串解析”和“字符串构建”是 C++ 最顺手的工具之一。**
